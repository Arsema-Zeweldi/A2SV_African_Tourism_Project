# Africa Tourism Intelligence Platform - PRD

## vision
Africa-First • Budget-Aware • AI-Assisted • Community-Driven • Mobile-First

## problem
Despite Africa’s vast cultural diversity, natural beauty, and unique travel experiences, intra-African tourism remains significantly underdeveloped.

## Core Features (V1)
- Interactive AI Planner
- Community Package Marketplace
- Streamer Tracks (e.g., IShowSpeed Tour)
- Budget-Friendly Options
- Travel Intelligence (Visa, Safety, Costs)

## Goals
- ≥5,000 registered users within first 2 months
- ≥30% of users create at least one itinerary
- ≥60% of itinerary creation flows completed
